# Poke App

## Comandos

### Run android

```cmd
npm run android
# or
yarn android
```
### Clean build

```cmd
npm run clean
# or
yarn clean
```

### Release APK

```cmd
npm run release
# or
yarn release
```

### Build APK

```cmd
npm run build:apk
# or
yarn build:apk
```
